Ranks in Bryxel Realm

On the Bryxel Realm, you are able to obtain ranks by earning rank points in-game by completing specific tasts, such as voting and playing games. To see all tasks that you are able to earn rank points through, please use this command in-game /jobs. To see all current rank features, please use the in-game command /rankup.
